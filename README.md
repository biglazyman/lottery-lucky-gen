# 🎰 欧气选号机 (Lucky Lotto Gen)

> **"把命运交给运气，而不是彩票站的机选。"**

## 🤔 这是什么？

这是一个专门帮“选择困难症”的彩民朋友生成号码的工具。

如果你不想研究走势图，又不信任彩票站自带的“机选”按钮，不妨试试这里的**欧气注入**。

## 💡 为什么用它？

1.  **拒绝“伪随机”**：彩票站的机选通常是写死的程序，这里我们使用更纯粹的随机方式，主打一个“随缘”。
2.  **自动同步**：每天自动更新最新的官方开奖数据，不用你去搜结果。
3.  **一键兑奖**：自动把你生成的号码和历史开奖记录比对，看看如果你早点买，能中多少钱（哪怕是云中奖也很爽）。
